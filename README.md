# Cellarium V1

Prototype PWA autonome de Cellarium, construit uniquement sur le lot :

- Accueil
- Cave
- Fiche bouteille

Les trois ecrans sont guides par les maquettes officielles dans `assets/reference` :

- `01 ACCUEIL REFERENCE.PNG`
- `02 CAVE REFERENCE.PNG`
- `03 FICHE BOUTEILLE REFERENCE.PNG`

## Utilisation

Ouvrez `index.html` dans un navigateur, ou publiez le dossier sur GitHub Pages.

Les donnees sont stockees localement dans `localStorage`.

## Fonctions incluses

- Entree depuis l'accueil vers la cave
- Recherche et filtres dans la cave
- Ajout manuel d'une bouteille
- Ouverture d'une fiche bouteille
- Stock `- / +`
- Note personnelle modifiable

Les autres ecrans ne sont pas developpes dans ce lot.
