# CELLARIUM — DESIGN SYSTEM OFFICIEL

## Source de vérité visuelle

Les images présentes dans ce dossier constituent la référence graphique officielle de Cellarium.

Elles définissent :
- l'identité visuelle ;
- les proportions ;
- la disposition des éléments ;
- les couleurs ;
- les textures ;
- les typographies ;
- l'ambiance générale.

## Principe fondamental

Cellarium n'est pas une application moderne générique.

Cellarium est une bibliothèque du goût et du temps.

Chaque écran doit évoquer :
- une bibliothèque ancienne ;
- une cave voûtée ;
- un carnet de voyage ;
- un cabinet de curiosités ;
- un patrimoine vivant.

## Règle de développement

Les fonctionnalités doivent être intégrées dans les maquettes.

Les maquettes ne doivent jamais être modifiées pour s'adapter à des composants techniques génériques.

En cas de conflit : LES MAQUETTES PRÉVALENT.

## Interdictions

Ne pas utiliser :
- Bootstrap par défaut
- Tailwind par défaut
- Material Design
- interfaces SaaS modernes
- cartes blanches génériques
- boutons modernes standardisés
- tableaux de bord impersonnels

## Philosophie Cellarium

Chaque bouteille est un livre.
Chaque dégustation est une page.
Chaque souvenir est un chapitre.

Le vin est le témoin.
Le temps est la véritable matière première.

## Objectif final

Construire une application qui ressemble davantage à une bibliothèque patrimoniale interactive qu'à un logiciel de gestion de cave.
Chaque décision de design doit servir cette ambition.
